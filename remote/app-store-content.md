## Description

ClawTab puts your Mac's AI agents in your pocket. Schedule, monitor, and control Claude Code sessions and automated jobs from anywhere - right from your phone.

Run AI coding agents on your Mac and never miss a question. When Claude needs your input, ClawTab detects it automatically and sends you a push notification with tappable answer cards. No more babysitting terminal windows.

KEY FEATURES

- Remote job control: Run, pause, resume, and stop jobs on your Mac from your phone
- Agent question detection: When Claude Code asks a question, answer it with a single tap
- Live log streaming: Watch terminal output in real time
- Cron scheduling: Schedule recurring jobs with standard cron expressions
- Push notifications: Get notified when jobs complete or need attention
- Multi-device support: Pair with multiple Macs and switch between them
- Workspace sharing: Grant colleagues access to monitor and control your agents
- Run history: Review past job runs, exit codes, and logs
- Three job types: Run scripts, Claude prompts, or project-based AI agents
- Secure by design: Secrets stay in your Mac's Keychain, never transmitted

HOW IT WORKS

1. Install the ClawTab desktop app on your Mac (free, open source)
2. Pair your phone by scanning the QR code or entering your device code
3. Create jobs - shell scripts, Claude Code prompts, or folder-based agents
4. Control everything from your phone: run jobs, stream logs, answer agent questions

Whether you're running a single Claude Code session or orchestrating dozens of AI agents across projects, ClawTab keeps you in control without being chained to your desk.

Self-hosting available for teams and enterprises who want to run their own relay server.

## App Review Notes

This app is a companion to the ClawTab macOS desktop application. It requires a Mac running ClawTab desktop to function. The app communicates with the desktop via a relay server (hosted at relay.clawtab.cc or self-hosted).

Demo mode is available without a paired desktop - the app shows sample job cards when no subscription is active.
